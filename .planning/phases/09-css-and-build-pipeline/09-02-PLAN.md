# Phase 9 / Plan 02

## Objective

Move the deployment and bootstrap workflow onto cargo-leptos so production builds no longer depend on Node/npm-era frontend steps.

## Implementation

1. Replace the manual release build script with a cargo-leptos release/precompress flow.
2. Update Docker to build the actual Leptos app/server output and copy `target/site` into the runtime image without Node.
3. Update bootstrap/install instructions to prepare the cargo-leptos + Tailwind standalone toolchain instead of requiring npm for the application build.

## Verification

- `cargo leptos build --release --precompress`
- `cargo check -p bominal-server --features ssr`
- `docker build` or `docker buildx build` with the updated context strategy, if Docker is available
