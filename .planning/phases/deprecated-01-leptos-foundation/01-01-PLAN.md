    **Step 2: Rewrite bominal-app/Cargo.toml**

    Replace the entire contents of `crates/bominal-app/Cargo.toml` with:

    ```toml

    name = "bominal-app"
    version.workspace = true
    edition.workspace = true

    [lib]
    crate-type = ["cdylib", "rlib"]

    [dependencies]
    leptos = { workspace = true, features = ["islands"] }
    leptos_meta.workspace = true
    leptos_router.workspace = true
    leptos_axum = { workspace = true, optional = true }

    bominal-domain = { path = "../bominal-domain", default-features = false }
    bominal-service = { path = "../bominal-service", optional = true }

    serde.workspace = true
    serde_json.workspace = true
    reqwest = { workspace = true, optional = true }

    # Client-only dependencies
    wasm-bindgen = "0.2"
    wasm-bindgen-futures = "0.4"
    web-sys = { version = "0.3", features = [
        "Document",
        "Element",
        "Event",
        "EventSource",
        "EventTarget",
        "Headers",
        "HtmlDocument",
        "HtmlElement",
        "Location",
        "Navigator",
        "Request",

    [features]
    default = []
    ssr = [
        "leptos/ssr",
        "leptos_meta/ssr",
        "leptos_router/ssr",
        "dep:leptos_axum",
        "dep:bominal-service",
        "dep:reqwest",
    ]
    hydrate = [
        "leptos/hydrate",
        "leptos_meta/hydrate",
        "leptos_router/hydrate",
    ```

    - Added `leptos_axum` as optional SSR dependency
    - Added `bominal-service` as optional SSR dependency
    - Added `reqwest` as optional SSR dependency (for server functions proxying to /api/)
    - Added `leptos_meta/ssr` and `leptos_router/ssr` to SSR features
    - Added `leptos_meta/hydrate` and `leptos_router/hydrate` to hydrate features

    Per FND-02: SSR/hydrate features with islands architecture.
