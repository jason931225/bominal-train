# Phase 3 / Plan 02

## Objective

Port the responsive navigation chrome and integrate it into the protected shell with route-aware active styling.

## Implementation

1. Add local `Sidebar` and `BottomNav` components in `bominal-app` using `use_location()` to compute visibility and active-state behavior.
2. Integrate the nav components into the protected layout from Plan 01 so desktop and mobile shells match the current product structure.
3. Extend `crates/bominal-app/style/app.css` with shell/navigation classes derived from the Bominal train visual language and the shared token bridge.
