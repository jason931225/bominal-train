# Phase 2 / Plan 01

## Objective

Replace the Phase 1 placeholder i18n layer with the real locale-aware app surface and port the shared frontend formatting helpers into `bominal-app`.

## Implementation

1. Add `bominal-domain` as a dependency of `bominal-app` and reuse its `Locale`, translation tables, and station-name registry instead of keeping app-local string maps.
2. Rework `crates/bominal-app/src/i18n.rs` to:
   - parse locale from the `bominal-locale` cookie during SSR,
   - expose a Leptos-context `current_locale()` helper,
   - provide `t()` and `t_locale()` wrappers on top of `bominal_domain::i18n`.
3. Add `crates/bominal-app/src/utils.rs` implementing the 8 shared formatting helpers currently present in the Svelte frontend.
4. Export the new modules from `crates/bominal-app/src/lib.rs`.

## Verification

- `cargo test -p bominal-app --lib`
- `cargo check -p bominal-app --features ssr`
- `cargo check -p bominal-app --target wasm32-unknown-unknown --features hydrate`
