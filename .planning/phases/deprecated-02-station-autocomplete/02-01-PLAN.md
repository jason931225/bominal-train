**Step 2 -- Create StationInput.svelte (GREEN).**

Create `frontend/src/lib/components/StationInput.svelte` implementing the full component per the skeleton in 02-RESEARCH.md. Key implementation details:

**Props interface (per D-01):**
```typescript
interface Props {
  value: string;       // $bindable() -- two-way bound station name
  label: string;       // "출발" or "도착"
  placeholder?: string; // "역 선택"
  provider: string;    // "SRT" or "KTX"
  name: string;        // "dep" or "arr" -- used for unique ARIA IDs
  onselect?: (match: SuggestMatch) => void; // optional callback
}
```

**State variables:**
- `suggestions: SuggestMatch[]` -- current suggestion list ($state)
- `expanded: boolean` -- dropdown visibility ($state)
- `activeIndex: number` -- keyboard highlight index, -1 = none ($state)
- `loading: boolean` -- API call in flight ($state)
- `isComposing: boolean` -- IME composition flag ($state, per D-02)
- `debounceTimer: ReturnType<typeof setTimeout> | undefined` -- debounce handle
- `lastQuery: string` -- for cache/re-focus behavior ($state, per D-06)
- `cachedSuggestions: SuggestMatch[]` -- cached results ($state, per D-06)

**ARIA IDs (per D-05, unique per instance via name prop):**
- Input id: `{name}-station-input`
- Listbox id: `{name}-station-listbox`
- Option ids: `{name}-station-option-{i}`
- `activeDescendant = $derived(activeIndex >= 0 ? \`${name}-station-option-${activeIndex}\` : undefined)`

**Debounce (per D-03):** 250ms timer. `scheduleFetch(query)` clears previous timer, returns early if `query.length < 1`, else sets 250ms timeout calling `fetchSuggestions(query)`.

**IME handling (per D-02):**
- `handleCompositionStart()`: sets `isComposing = true`
- `handleCompositionEnd()`: sets `isComposing = false`, calls `scheduleFetch(value)`
- `handleInput()`: updates `value`, returns early if `isComposing === true`, else calls `scheduleFetch(value)`

**Keyboard navigation (per D-04):**
- ArrowDown: `activeIndex = (activeIndex + 1) % suggestions.length`, call `scrollActiveIntoView()`, `e.preventDefault()`
- ArrowUp: `activeIndex = activeIndex <= 0 ? suggestions.length - 1 : activeIndex - 1`, `scrollActiveIntoView()`, `e.preventDefault()`
- Enter: if `activeIndex >= 0`, call `selectStation(suggestions[activeIndex])`, `e.preventDefault()`
- Escape: set `expanded = false`, `activeIndex = -1`, `e.preventDefault()`
- If dropdown closed and ArrowDown pressed with suggestions available: open dropdown, set `activeIndex = 0`

**Stale response guard:** In `fetchSuggestions`, after `await suggestStations(provider, query)`, check `if (query !== value) return;` to discard stale responses.

**Focus/blur (per D-06):**
- `handleFocus()`: if `cachedSuggestions.length > 0 && value === lastQuery`, reshow cached suggestions
- `handleBlur()`: `setTimeout(() => { expanded = false; activeIndex = -1; }, 200)` (200ms delay for mousedown to fire)

**Provider change:** Add `$effect` watching `provider` -- when it changes, clear `cachedSuggestions`, `suggestions`, set `expanded = false`, `activeIndex = -1`.

**Reset activeIndex on suggestions change:** Add `$effect` watching `suggestions` -- when length changes, reset `activeIndex = -1`.

**scrollActiveIntoView():** `requestAnimationFrame(() => { document.getElementById(\`${name}-station-option-${activeIndex}\`)?.scrollIntoView({ block: 'nearest' }); })`

**selectStation(match):** Set `value = match.name_ko`, `expanded = false`, `activeIndex = -1`, call `onselect?.(match)`.

**Template structure (per UI-SPEC):**
```svelte
<div class="flex-1 relative">
  <label class="text-xs font-medium mb-1 block" style="color: var(--color-text-tertiary)" for="{name}-station-input">
    {label}
  </label>
  <div class="relative">
    <input
      id="{name}-station-input"
      type="text"
      role="combobox"
      aria-expanded={expanded}
      aria-controls={listboxId}
      aria-activedescendant={activeDescendant}
      aria-autocomplete="list"
      class="w-full rounded-xl px-3 py-2.5 text-sm font-medium outline-none"
      style="background: var(--color-bg-sunken); color: var(--color-text-primary); border: 1px solid var(--color-border-default)"
      placeholder={placeholder}
      value={value}
      oninput={handleInput}
      onkeydown={handleKeydown}
      onfocus={handleFocus}
      onblur={handleBlur}
      oncompositionstart={handleCompositionStart}
      oncompositionend={handleCompositionEnd}
    />
    {#if loading}
      <div class="absolute right-3 top-1/2 -translate-y-1/2" aria-label={loadingLabel}>
        <svg class="animate-spin w-4 h-4" viewBox="0 0 24 24" fill="none">
          <circle cx="12" cy="12" r="10" stroke="var(--color-brand-text)" stroke-width="2" opacity="0.25" />
          <path d="M12 2a10 10 0 0 1 10 10" stroke="var(--color-brand-text)" stroke-width="2" stroke-linecap="round" />
        </svg>
      </div>
    {/if}
  </div>
  {#if expanded}
    <ul id={listboxId} role="listbox" class="absolute z-20 left-0 right-0 mt-1 glass-panel rounded-xl overflow-hidden max-h-48 overflow-y-auto">
      {#if suggestions.length === 0}
        <li class="px-3 py-2.5 text-sm" style="color: var(--color-text-tertiary)">
          {noMatchText}
        </li>
      {:else}
        {#each suggestions as match, i}
          <li id="{name}-station-option-{i}" role="option" aria-selected={i === activeIndex}>
            <button type="button" class="w-full text-left px-3 py-2.5 text-sm transition-colors"
              class:bg-[var(--color-interactive-hover)]={i === activeIndex}
              style="color: var(--color-text-primary)"
              onmousedown={() => selectStation(match)}>
              {match.name_ko}
              <span class="text-xs ml-1" style="color: var(--color-text-tertiary)">{match.name_en}</span>
            </button>
          </li>
        {/each}
      {/if}
    </ul>
  {/if}
</div>
```
