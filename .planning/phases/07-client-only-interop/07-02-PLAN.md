# Phase 7 / Plan 02

## Objective

Consolidate the Evervault and passkey browser bridges into one app-local runtime asset and keep the Rust side as the stable interop boundary.

## Implementation

1. Merge the passkey ceremony helpers into `crates/bominal-app/assets/interop.js`, preserving the existing Evervault card helpers.
2. Update `crates/bominal-app/index.html` to load only the unified interop asset.
3. Remove the legacy split passkey asset once the unified runtime surface is in place and the Rust wrappers still compile in both SSR and hydrate builds.
