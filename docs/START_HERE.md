# Start Here

This docs set is split by audience so policy and procedures do not drift.

Pick your path:
- Humans: `docs/humans/README.md`
- Agents: `AGENTS.md`, then `docs/agents/README.md`

## Taxonomy

- Governance (canonical policy): `docs/governance/**`
- Humans (procedures/reference): `docs/humans/**`
- Agents (mandatory overlays): `docs/agents/**`
- Pointer library: `docs/README.md`
- Intent routing: `docs/INTENT_ROUTING.md`

## Anti-Drift Rule

- Policy must live in one canonical governance file.
- Human docs link to policy; they do not redefine policy.
- Agent docs contain agent-specific deltas and hard constraints.
