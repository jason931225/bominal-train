# Start Here

The documentation system uses canonical policy by domain.

Read in this order:
1. `docs/MANUAL.md`
2. `docs/GITHUB_GOVERNANCE.md`
3. `docs/CI_CD_POLICY.md`
4. `docs/README.md`
5. `AGENTS.md` (for automated contributors)

Historical migration-era docs were intentionally removed from the primary path.
`docs/handoff/**` remains available as preserved reference material.
