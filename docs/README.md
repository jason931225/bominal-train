# Bominal Docs

This folder is the operational and engineering documentation pack for Bominal.

1. `ARCHITECTURE.md` - system design and module boundaries
2. `CONTRIBUTING.md` - local setup, coding standards, and testing
3. `DEPLOYMENT.md` - production deployment and environment separation
4. `RUNBOOK.md` - operations, troubleshooting, and recovery steps
5. `SECURITY.md` - security controls, secret handling, and hardening priorities

For agent-focused instructions, also read the root `AGENTS.md`.
