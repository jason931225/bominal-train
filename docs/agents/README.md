# Agents Docs

These are mandatory overlays for automated contributors.

## Required Order

1. `AGENTS.md`
2. `docs/agents/GUARDRAILS.md`
3. `docs/agents/PERMISSIONS.md`
4. `docs/agents/EXECUTION_PROTOCOL.md`
5. `docs/INTENT_ROUTING.md`
6. Canonical docs from `docs/README.md`

## Scope

- `GUARDRAILS.md`: hard constraints
- `PERMISSIONS.md`: approval gates and risk tiers
- `EXECUTION_PROTOCOL.md`: agent execution workflow
- `DEPLOYMENT.md`: deploy overlay
- `DOCUMENTATION.md`: docs workflow overlay
